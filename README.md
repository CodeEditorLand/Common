# 👨🏻‍🏭 [Common] —

## Installation

First, install the `Common` component like so:

```sh
npm install -D -E @codeeditorland/common
```

**`Index.ts`**

```ts
import {
	Access,
	Decrypt,
	Encrypt,
	Get,
	Put,
	Redirect,
	Response,
	Uint8ArrayFromObject,
} from "@codeeditorland/common";
```
